# Patient handout — Fizioterapia ime

Route:

- `/patient-handout`

This page is designed to be printed or saved as PDF by the clinic/physiotherapist.

## What it explains

- Patient uses the platform only with username + code from physiotherapist.
- The exercise plan is created only by the physiotherapist.
- Patient should do exercises slowly and safely.
- Patient should submit pain score 0–10.
- Pain 7/10 or higher means stop and contact physiotherapist.
- AI Movement Check gives feedback only.
- AI does not diagnose.
- AI does not replace the physiotherapist.
- Camera video is not stored in MVP.

## Fields included

The printable handout includes blank fields for:

- Username
- Code
- Physiotherapist

## Print instructions

Open `/patient-handout` in browser and use:

- iPhone/iPad: Share → Print or Save to Files as PDF
- Desktop: Cmd/Ctrl + P → Save as PDF

## Safety text

Do not remove the stop rules or AI disclaimer before clinical/legal review.
