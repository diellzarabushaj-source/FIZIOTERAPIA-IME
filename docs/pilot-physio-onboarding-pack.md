# Pilot physiotherapist onboarding pack

Use this pack to invite and onboard the first test physiotherapist for Fizioterapia ime.

## Pilot scope

Start small:

- 1 physiotherapist
- 1 to 3 test patients
- 3 to 7 days of testing
- no mass launch
- no public medical claims

## What the physiotherapist will test

- Login to Physiotherapist Portal
- Subscription/access status
- Create a patient
- Generate username and patient code
- Assign exercises
- Review patient progress
- Review pain scores
- Review AI Movement Check feedback
- Open or print patient report

## Important clinical rule

AI Movement Check gives movement-quality feedback only. It does not diagnose, does not prescribe, and does not replace the physiotherapist.

If the patient reports pain 7/10 or higher, the patient should stop the exercise and contact the physiotherapist.

## Feedback questions

Ask after 3 to 7 days:

1. Was patient creation clear?
2. Was exercise assignment easy?
3. Was username + code clear for the patient?
4. Was the AI disclaimer clear enough?
5. Was the PDF report useful?
6. What would stop you from using this with a real patient?
7. What should be simplified?
8. What feature is missing before payment?
